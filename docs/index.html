NEXUS Agent — Installation

1 Voraussetzung: Python 3.10+

Python muss installiert sein. Download: python.org

2 PowerShell als Administrator starten

Rechtsklick auf PowerShell → "Als Administrator ausführen"

3 Installer ausführen 
# Standard-Installation (Brain IP: 192.168.178.202)
irm https://werizu.github.io/nexus/install.ps1 | iex

# Oder mit eigener Brain-IP:
$env:NEXUS_BRAIN="192.168.178.100"; irm https://werizu.github.io/nexus/install.ps1 | iex
4 Fertig!

Der Agent läuft als Windows-Dienst und verbindet sich automatisch mit dem NEXUS Brain.

NEXUS Mac Agent — Installation

1 Voraussetzung: Python 3.10+

Bereits auf macOS vorhanden, oder: brew install python3

2 Terminal öffnen und Installer ausführen 
# Standard-Installation (Brain IP: 192.168.178.202)
curl -fsSL https://werizu.github.io/nexus/install-mac.sh | bash

# Oder mit eigener Brain-IP:
NEXUS_BRAIN="192.168.178.100" curl -fsSL https://werizu.github.io/nexus/install-mac.sh | bash
3 Fertig!

Der Agent läuft als LaunchAgent und startet automatisch bei Login.

Agent-Befehle

Windows

# Service-Status prüfen
Get-Service NexusAgent

# Service stoppen / starten
Stop-Service NexusAgent
Start-Service NexusAgent

# Agent deinstallieren
python "C:\Program Files\NEXUS Agent\nexus_service.py" remove

# Logs anzeigen
Get-Content "C:\Program Files\NEXUS Agent\nexus-agent.log" -Tail 20

macOS

# Status prüfen
launchctl list | grep nexus

# Agent stoppen / starten
launchctl unload ~/Library/LaunchAgents/com.nexus.agent.plist
launchctl load ~/Library/LaunchAgents/com.nexus.agent.plist

# Agent deinstallieren
launchctl unload ~/Library/LaunchAgents/com.nexus.agent.plist
rm -rf ~/.nexus-agent ~/Library/LaunchAgents/com.nexus.agent.plist

# Logs anzeigen
tail -f ~/.nexus-agent/nexus-agent-mac.log
